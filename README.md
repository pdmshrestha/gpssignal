# GPS Signal
Get information about your location from GPS satellites with speedometer. 
Now you can know the speed of bus you are travelling. 
You can also copy the GPS data in clipboard and paste it anywhere. 
Address of your current location is obtained from GOOGLE SERVER, you need a data connection for this. 

This application was developed for fun purpose and hope it will be helpful for you guys too.

Demo here: https://play.google.com/store/apps/details?id=com.pdm.gpssignal
